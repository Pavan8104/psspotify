# PSSpotify - Professional Frontend Music Player

## ✨ Features
- Vite-powered, production build ready
- 10 NCS tracks with covers/audio
- Dynamic song list
- Play/pause, next/prev, seekbar, volume
- **localStorage persistence**: Recently played songs survive refresh
- Responsive design, mobile-friendly
- Search songs

## 🚀 Quick Start
```bash
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)

## 📦 Build for Production
```bash
npm run build
```
Serve `dist/` folder.

## 🛠 Tech Stack
- Vanilla JS + Vite (5KB bundle)
- FontAwesome icons
- localStorage (frontend DB)

## 🎵 Tracks
Best of NCS Release - Warriyo, Cielo, DEAF KEV etc.
